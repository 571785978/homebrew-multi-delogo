# Wernerturing Multi-delogo

## How do I install these formulae?
`brew install wernerturing/multi-delogo/<formula>`

Or `brew tap wernerturing/multi-delogo` and then `brew install <formula>`.

Or install via URL (which will not receive updates):

```
brew install https://raw.githubusercontent.com/wernerturing/homebrew-multi-delogo/master/Formula/<formula>.rb
```

## Documentation
`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
